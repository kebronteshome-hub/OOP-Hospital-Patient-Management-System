## Description
This is a simple hospital patient management system i made using Java programming and OOP concepts as we have learnt. This system handles different types of patients like in-patients and out-patients.

## What this project does
- Add patient details
- Store patient information
- Show patient details
- Manage different types of patients using inheritance concept

## OOP concepts used
- Encapsulation 
- Inheritance (InPatient and OutPatient classes extend Patient class)
- Polymorphism (method overriding in displayInfo )

## Project structure
- Patient.java (main/base class)
- InPatient.java (for admitted patients only )
- OutPatient.java (for visiting patients)
- Hospital.java (handles patient list)
- Main.java (runs the entire accurate program)

## How to run

2. Run Main.java
